Privacy Policy for LangTest Pro Chrome Extension

Effective Date: 02/03/2025

Introduction

This Privacy Policy describes how the "LangTest Pro" Chrome extension (the "Extension") collects, uses, and protects user data. By installing and using the Extension, you agree to the terms of this Privacy Policy.

Information We Collect

  • Website URLs: The Extension sends the URLs of the web pages you are viewing to our remote server (hosted on Heroku) for language analysis.
  • Text Content: The Extension uses Selenium on a remote server to extract text content from the URLs provided. This text is then processed to detect languages.

How We Use Information

  • Language Analysis: The primary purpose of the Extension is to analyze the language composition of web pages. The data collected is used to identify and count the languages present on the page.
  • Service Improvement: We may use aggregated, anonymized data to improve the performance and accuracy of the Extension.
  • Server Maintenance: Server logs are used for debugging and maintenance purposes.

Data Sharing

  • Third-Party Services: We use Heroku to host our remote server. Heroku's privacy policy applies to the data processed on their platform.
  • No Personal Information: We do not collect or share any personally identifiable information (PII) other than the website URLs and their text content necessary for the function of the extension.
  • No Selling Data: We do not sell or rent user data to third parties.

Data Security

  • Secure Transmission: Data transmitted between the Extension and our server is encrypted using HTTPS.
  • Server Security: We implement reasonable security measures to protect our server from unauthorized access.

User Rights

  • Data Access: You can request access to the data we have collected about you.
  • Data Deletion: You can request that we delete the data we have collected about you.
